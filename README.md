# Личный финансовый кошелек - Android приложение

Android клиент для системы управления личными финансами на Spring Boot.

## Требования

- Android Studio Arctic Fox или новее
- Android SDK 24+ (Android 7.0)
- Kotlin 1.9+

## Настройка

1. Откройте проект в Android Studio
2. Убедитесь, что Spring Boot сервер запущен (см. `kursovaya/README_NETWORK.md`)

### Настройка сети

Приложение автоматически определяет тип устройства и использует соответствующий адрес сервера:

- **Для эмулятора:** Автоматически используется `http://10.0.2.2:8080/`
- **Для физического устройства:** Необходимо указать IP адрес вашего компьютера:
  1. Узнайте IP адрес вашего компьютера в локальной сети (Windows: `ipconfig`, Linux/Mac: `ifconfig`)
  2. Откройте `app/src/main/java/ru/labs/wallet/data/api/NetworkUtils.kt`
  3. Измените значение константы `PHYSICAL_DEVICE_SERVER_IP` на IP вашего компьютера
  4. Пересоберите приложение

**Важно:** Убедитесь, что Spring Boot сервер настроен для работы на всех интерфейсах (в `application.properties` должно быть `server.address=0.0.0.0`)

## Структура проекта

```
app/src/main/java/ru/labs/wallet/
├── data/
│   ├── api/           # Retrofit API интерфейсы
│   ├── model/         # Модели данных
│   ├── repository/    # Репозитории для работы с API
│   └── local/         # Локальное хранилище (SharedPreferences)
├── ui/
│   ├── adapter/       # Адаптеры для RecyclerView
│   ├── viewmodel/     # ViewModels
│   └── *.kt           # Activities и Fragments
```

## Основные функции

- Регистрация и авторизация пользователей
- Просмотр операций (доходы/расходы)
- Добавление новых операций
- Управление категориями
- Управление бюджетом

## Использование

1. Запустите Spring Boot сервер из папки `kursovaya`
2. Запустите Android приложение
3. Зарегистрируйтесь или войдите с существующими учетными данными
4. Начните управлять своими финансами!

## API Endpoints

Приложение использует следующие API endpoints:

- `/api/auth/login` - Вход
- `/api/auth/register` - Регистрация
- `/api/operations/me` - Получить операции текущего пользователя
- `/api/categories/me` - Получить категории текущего пользователя
- `/api/budgets/me` - Получить бюджет текущего пользователя

Все запросы используют cookie-based аутентификацию через Spring Security.
